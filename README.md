# TP 6 : Découverte des Hooks Avancés et des Hooks Personnalisés

# Objectifs : 
 - Utilisation de useReducer pour structurer l’état  
 - Manipulation du DOM et stock des valeurs persistantes avec useRef  
 - Création et utilisation d'un Hook personnalisé (useFetch)  
 - Gestion des effets avec nettoyage grâce à useEffect

# Technologies utilisées :
- node v24.14.0
- npm  11.9.0
- VS Code
- React
- Vite
